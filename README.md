# 短網址產生器

## 功能描述

+ 使用者可以輸入一組網址之後可以得到由5個英數字取代的短網址。

+ 使用者點擊連結可直接跳轉頁面。

+ 使用者得到網址之後可以透過下方按鈕直接複製。

## 安裝

1. 開啟終端機輸入以下指令以clone此專案並執行

```
git clone https://github.com/Tzuminn/shortenURL.git
```

2. 移動到專案資料夾

```
cd shortenURL
```

3. 安裝npm套件

```
npm install
```

4. 開啟專案

```
npm run dev
```

當終端機顯示 server is running on http://localhost:3000 ，表示執行成功。
